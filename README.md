## Usage
There are multiple ways to use the Notes Application, please choose the one you desire. 

For the standard CLI application, please run the following to get an understanding of how to run the program. 
```powershell
go run . app help
```
For the API serviced version, the routing will be slightly different with improved features. Run the following to understand how to use the enhanced application. 
```powershell
go run ./cmd/cli app help
```
## Future Updates

In the API version of the Information Entry System, right now it is used as a notes application where the user can create, delete and view notes. Additional features will be added where more informational entries such as tasks with due dates and different priorities and lists with multiple items. Furthermore with the additional application types, there will be editing, sorting and viewing features to make it more user friendly.
